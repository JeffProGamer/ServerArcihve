
Minecraft Beta 1.0.2 Server Control Panel (By Jeff_pro_gamer_351)

Instructions:
1. Place 'beta_server.jar' (Minecraft Beta 1.0.2 server file) in this folder.
2. Open terminal/cmd in this folder.
3. Run the following commands:
   npm init -y
   npm install express
   node server.js
4. Use Ngrok to expose it online:
   ngrok http 3000
5. Replace 'https://YOUR-NGROK-URL' in index.html with the generated Ngrok URL.
6. Open index.html in your browser to control your server!
